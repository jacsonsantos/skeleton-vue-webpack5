# Skeleton Vue and Webpack 5

### Packages
- VueJs
- VueX
- VueRouter
- Webpack
- Sass

### Creating folders dist

```
mkdir -p public/assets/dist/css
mkdir -p public/assets/dist/js
```

### Building code

```
npm run start
```

### Starting server

```
npm run server
```
